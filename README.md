# MealPal

## Introduction
MealPal là ứng dụng di động hỗ trợ người dùng lập kế hoạch bữa ăn cá nhân hóa, tối ưu nguyên liệu sẵn có, và theo dõi dinh dưỡng hàng ngày/tuần. Ứng dụng hướng tới việc giảm thời gian chuẩn bị, tránh lãng phí thực phẩm, và xây dựng thói quen ăn uống lành mạnh.

---

## Getting Started

### Clone repository
```bash
git clone https://github.com/evgo-team/backend.git
cd backend
```

### Run with Docker
```bash
docker compose up -d --build
```
